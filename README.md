# ML-Based Data Leak Detection System

An AI-powered system to detect data leaks by comparing a master database with a current database.

## Features
- Automatic PII detection
- Record deletion & injection detection
- Feature engineering for data differences
- Isolation Forest anomaly detection
- Risk categorization (Low / Medium / High)
- Interactive Streamlit dashboard
- Downloadable CSV report

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Isolation Forest, Streamlit

## Use Cases
- Data security auditing
- Compliance monitoring
- Enterprise database change detection
